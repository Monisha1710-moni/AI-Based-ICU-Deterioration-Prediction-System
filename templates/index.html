About This Project

This AI-powered healthcare application predicts ICU patient deterioration using Machine Learning and Explainable AI.

The model uses Age, Glucose, Heart Rate, Temperature, Systolic Blood Pressure and Respiratory Rate.

Built using Python, Flask, XGBoost, SHAP, HTML, CSS and JavaScript.
